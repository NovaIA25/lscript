# HTML ou CSS : quelle différence ?

Quand on parle de créer un site web, on entend toujours parler de HTML et CSS. Mais c'est quoi la différence ?

## HTML = la structure

Le HTML, c'est le **squelette** de ta page web. Il définit ce qu'il y a sur la page :

- Un titre
- Un paragraphe
- Une image
- Un bouton

C'est comme les murs et les pièces d'une maison. On sait qu'il y a une cuisine, un salon, une chambre... mais on ne sait pas encore à quoi ça ressemble.

## CSS = l'apparence

Le CSS, c'est la **décoration**. Il définit à quoi ça ressemble :

- La couleur du texte
- La taille des titres
- L'espacement entre les éléments
- La police d'écriture

C'est comme la peinture, les meubles et la déco de ta maison.

## Un exemple simple

**HTML (ce qu'il y a) :**

```html
<h1>Bienvenue sur mon site</h1>
<p>Ceci est un paragraphe.</p>
```

**CSS (à quoi ça ressemble) :**

```css
h1 {
  color: blue;
  font-size: 32px;
}
```

## Pourquoi les séparer ?

Parce que c'est plus simple à gérer. Tu peux changer l'apparence de ton site (couleurs, tailles) sans toucher à la structure (textes, images).

## Lequel apprendre en premier ?

**HTML.** Parce que sans structure, il n'y a rien à décorer.

Mais tu apprendras vite les deux ensemble, ils vont toujours de pair.

---

**Retiens :** HTML = ce qu'il y a sur la page. CSS = à quoi ça ressemble.
